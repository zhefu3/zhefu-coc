## 关于 AI 守秘人（可选）

游戏主体（场景、选项、检定、骰子、NPC、剧情）**全部静态、无需联网**，谁都能免费玩。

每作另有可选的「自由输入 · AI 守秘人」：你可以打字说出任何行动，由 Claude 即兴接住并推进。它需要你**自备 Anthropic API Key**——Key 只存在你自己浏览器的 localStorage 里，游戏本身不含也不上传任何密钥，费用记在你自己的账上。不填 Key 不影响正常游玩。

## 技术

单文件纯静态网页，零依赖、零构建：立绘为代码绘制的 SVG、音效为 Web Audio 实时合成、骰子为 CSS 3D。可直接用任意静态服务器托管。
